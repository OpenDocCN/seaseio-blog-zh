// first talk

利用动态规划和长跳转改进Top-K检索算法

现代搜索引擎必须跟上用户提交的文档和查询数量的巨大增长。要处理的问题之一是为给定的查询找到最佳的k个相关文档。这种操作必须很快，只有通过使用专门的技术才有可能。
Block max wand是解决这个问题的最著名的算法之一，其排名没有任何有效性下降。
在简单介绍之后，在本次演讲中，我将展示“使用可变大小块的更快BlockMax WAND”(SIGIR 2017)中介绍的一种策略，该策略应用于block max WAND数据，使算法执行速度提高了近2倍。
然后，将呈现BlockMaxWand算法的另一种优化(“更快的BlockMax WAND和更长的跳跃”，ECIR 2019)，用于减少短查询的时间执行。
