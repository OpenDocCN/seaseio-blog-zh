// first talk

查询松弛——一种介于搜索和推荐之间的重写技术

在搜索质量优化中，使用各种技术来提高召回率，尤其是为了避免空的搜索结果集。在大多数解决方案中，例如拼写校正和查询扩展，搜索查询被修改，而原始查询意图通常被保留。在我的演讲中，我将描述我用不同的方法来放松查询的实验。查询松弛是一种查询重写技术，它从多术语查询中删除一个或多个术语，否则将导致零结果。在许多情况下，查询词的移除需要改变查询意图，使得难以判断重写的查询的质量，从而难以决定应该移除哪个查询词。我认为，如果将查询放松视为搜索和推荐之间的一种技术，它可能会得到最好的理解。我的重点是一个基于使用Word2Vec嵌入的电子商务搜索环境中的解决方案。
